# Sustainable Airline Reservation System (C++)

## Overview
This project focuses on designing and implementing a more realistic Airline Reservation System using C++. Unlike basic booking systems, this project aims to simulate real-world airline operations by incorporating dynamic pricing, route optimization, and sustainability considerations.

The system handles core functionalities such as flight management, seat allocation, and booking, while also introducing algorithmic decision-making to improve efficiency and realism.

## Features
- Flight management (add, view, and manage flights)
- Seat allocation and booking system
- Prevention of double booking
- Dynamic pricing based on demand and seat availability
- Route optimization using graph-based algorithms
- Basic carbon footprint estimation for different routes

## Tech Stack
- C++
- Object-Oriented Programming (OOP)
- Data Structures and Algorithms (Graphs, Optimization)

## System Design
The system is structured into multiple modules:
- **Flight Module** – stores and manages flight data
- **Booking Module** – handles reservations and seat allocation
- **Pricing Module** – calculates ticket prices dynamically
- **Graph Module** – computes optimal routes between locations
